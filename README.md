## Getting started:

To run this locally please make sure you have installed all the needed dependencies:

```
yarn
```

or

```
npm install
```

Then go ahead and run:

```
yarn dev
```

or

```
npm run dev
```

## About this project:

This is a front-end coding challenge for https://stuart.com

To get it working right, make sure you provide valid addresses such as:

- 29 Rue du 4 Septembre
- 15 Rue de Bourgogne

Then go ahead and click on `Create job` to queue a new job. If all goes right, you should see a successful message on the right side of the screen
